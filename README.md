# Adidas Sales MIS Dashboard

## Overview
This project demonstrates an automated Sales MIS (Management Information System) dashboard built using Excel and Power Query for retail sales analysis.

The objective is to convert raw, messy sales data into clean, structured tables and generate business-ready insights with minimal manual effort.

---

## Business Problem
Retail sales data often comes in unstructured formats, leading to:
- Manual data cleaning
- Reporting errors
- Time-consuming analysis
- Lack of clear performance visibility

---

## Solution
An automated MIS system was built to:
- Clean raw sales data using Power Query
- Standardize formats and remove inconsistencies
- Generate sales, profit, and trend insights
- Enable one-click data refresh

---

## Key Metrics
- Total Sales
- Units Sold
- Operating Profit
- Operating Margin %
- Monthly & Yearly Sales Trend
- Region-wise Performance
- Sales Channel Analysis

---

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- (Optional) Power Pivot & DAX
